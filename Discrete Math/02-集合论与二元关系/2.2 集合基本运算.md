# 集合基本运算

## 基本运算定义

集合的基本运算包括：

- **并运算**（Union）：$A \cup B = \{x \mid x \in A \lor x \in B\}$
- **交运算**（Intersection）：$A \cap B = \{x \mid x \in A \land x \in B\}$
- **差运算**（Difference）：$A - B = \{x \mid x \in A \land x \notin B\}$
- **补运算**（Complement）：$\sim A = U - A$（相对于全集的差）
- **对称差**（Symmetric Difference）：$A \oplus B = (A - B) \cup (B - A)$

此处注意，二类运算是从右向左进行。

## 运算律

集合运算满足以下基本运算律：

- **幂等律**：$A \cup A = A$, $A \cap A = A$
- **交换律**：$A \cup B = B \cup A$, $A \cap B = B \cap A$
- **结合律**：$(A \cup B) \cup C = A \cup (B \cup C)$, $(A \cap B) \cap C = A \cap (B \cap C)$
- **分配律**：$A \cup (B \cap C) = (A \cup B) \cap (A \cup C)$, $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$
- **吸收律**：$A \cup (A \cap B) = A$, $A \cap (A \cup B) = A$
- **德摩根律**：$\sim(A \cup B) = \sim A \cap \sim B$, $\sim(A \cap B) = \sim A \cup \sim B$

## 包含关系与运算

包含的两侧可以进行同时的并或交运算，只要新的参与运算的集合也满足同方向的包含关系。

## 对称差的性质

注意这里对称差的一个运算性质：两边有相同元素时，可以利用对称差的交换律和结合律，同时对相同元素做对称差，消去相同元素。即 $A = B \Leftrightarrow A + C = B + C$，这里用 $+$ 代表对称差。
