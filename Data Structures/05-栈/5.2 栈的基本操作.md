# 栈的基本操作

栈作为一种线性数据结构，提供了一套标准的操作接口。以下是栈的核心操作：

## 操作列表

| 操作 | 描述 |
|------|------|
| **init** | 初始化栈，创建一个空栈 |
| **destroy** | 销毁栈，释放栈所占用的所有内存资源 |
| **isEmpty** | 判断栈是否为空，返回布尔值 |
| **length** | 返回栈中元素的个数 |
| **clear** | 清空栈中的所有元素，使栈恢复为空状态 |
| **push** | 入栈，将元素压入栈顶 |
| **pop** | 出栈，移除并返回栈顶元素 |
| **getTop** | 获取栈顶元素的值，但不将其从栈中移除 |

## 操作说明

- `push` 和 `pop` 是栈最核心的两个操作，体现了 LIFO 的特性
- `getTop` 与 `pop` 的区别在于是否删除栈顶元素
- `isEmpty` 常用于在执行 `pop` 或 `getTop` 之前检查栈是否为空，防止操作异常
