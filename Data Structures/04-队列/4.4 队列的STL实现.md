# 队列的STL实现

C++ 的标准模板库（STL, Standard Template Library）提供了现成的队列容器 `std::queue`，可以直接使用而无需手动实现底层细节。

## 头文件

```cpp
#include <queue>
```

## 基本用法

```cpp
std::queue<int> q;

q.push(1);       // 入队
q.push(2);
q.pop();         // 出队
int front = q.front();  // 访问队首元素
int back  = q.back();   // 访问队尾元素
bool empty = q.empty(); // 判空
size_t size = q.size(); // 获取元素个数
```

## 特点

- STL 队列默认基于 `deque`（双端队列）实现，也可指定底层容器为 `list`
- 提供了标准队列操作接口，简单高效
- 适用于大多数需要 FIFO 数据结构的应用场景
