## 👀 阅读方式

本仓库提供两种阅读模式：

### Obsidian（推荐）

各学科按 **章节组织为独立文件**，适合在 Obsidian 中打开整个仓库作为 Vault，利用知识图谱、双向链接、图谱可视化等功能浏览：

```

├── CSAPP.md                     ← 根节点（入口）
├── 01-计算机系统漫游/

│   ├── 1.1 信息就是位+上下文.md  ← 原子笔记
│   └── ...
├── 02-信息的表示和处理/
│   └── ...
└── ...
```

各学科 `notes/` 文件夹中包含 **传统整章笔记**，适合在 VSCode 中直接阅读 markdown 文件：


## 📝 笔记格式

结构化笔记遵循 [知识图谱构建方法论](知识图谱构建方法论.md) 中的规范：

- **原子化**：每个笔记只讲一个概念
- **树状结构**：根 → 章节 Hub → 原子笔记
- **统一命名**：原子笔记带 `X.Y` 数字前缀
- **Front Matter**：每篇笔记包含 `created`、`tags` 元数据
