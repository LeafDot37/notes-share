# 稳态分布

稳态分布是马尔可夫链理论中的重要概念。对于一个平稳马尔可夫信源，设其状态转移概率矩阵为 $\boldsymbol{P}$，其中元素 $p_{ij} = P(X_{n+1} = j \mid X_n = i)$。若存在一个概率分布向量 $\boldsymbol{\pi} = (\pi_1, \pi_2, \dots, \pi_r)$ 满足 $\boldsymbol{\pi} \boldsymbol{P} = \boldsymbol{\pi}$，且 $\sum_{i=1}^{r} \pi_i = 1$，则称 $\boldsymbol{\pi}$ 为该马尔可夫链的稳态分布。其意义是：一旦系统进入该分布，之后所有时刻的分布都将保持不变。

求马尔可夫链的平稳分布，即求转移矩阵 $\boldsymbol{P}$ 对应于特征值 $\lambda = 1$ 的左特征向量。由于 $\boldsymbol{P}$ 每一行元素之和为 $1$，根据佩龙-弗罗贝尼乌斯定理，其最大特征值的模长必然为 $1$，故特征值 $1$ 一定存在。实际计算时，由于分布向量为左向量不便处理，可将 $\boldsymbol{P} - \boldsymbol{I}$ 转置后转化为右向量问题，再通过高斯消元求解，最后利用归一化条件得到唯一的稳态分布。

稳态分布与极限熵密切相关。当马尔可夫链达到稳态时，极限熵即状态转移时的条件熵：

$$
H_\infty = H(X_{m+1} \mid X_1, \dots, X_m) = -\sum_{i=1}^{r} \pi_i \sum_{j=1}^{r} p_{ij} \log p_{ij}
$$

其中 $\pi_i$ 为稳态分布中状态 $i$ 的概率，$p_{ij}$ 为从 $i$ 转移到 $j$ 的概率。需要注意的是，存在稳态分布并不保证系统的遍历性——例如两个状态 $A$ 和 $B$ 周期性地互相转移，$\boldsymbol{\pi} = [0.5, 0.5]$ 是其稳态分布，但系统不满足遍历性，因为其不具有非周期性。遍历性要求从任意状态出发经过足够多步后都能到达其他任意状态，且出现状态的极限概率与初始状态无关。
