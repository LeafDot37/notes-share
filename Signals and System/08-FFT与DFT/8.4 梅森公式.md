# 梅森公式

## 从信号流图求系统函数

梅森公式（Mason's Formula）提供了从信号流图直接求解系统函数的通用方法，无需逐步化简流图。

## 公式定义

$$H(s) = \frac{Y(s)}{F(s)} = \frac{1}{\Delta} \sum_{k} P_k \Delta_k$$

其中：
- $H(s)$：系统函数（增益）
- $P_k$：第 $k$ 条前向通路的增益
- $\Delta$：特征式（流图的行列式）
- $\Delta_k$：第 $k$ 条前向通路对应的余子式

## 特征式 $\Delta$ 的展开

$$\Delta = 1 - \sum L_a + \sum L_b L_c - \sum L_d L_e L_f + \cdots$$

| 项 | 含义 |
|----|------|
| $\sum L_a$ | 所有不同回路的增益之和 |
| $\sum L_b L_c$ | 所有两两不接触回路的增益乘积之和 |
| $\sum L_d L_e L_f$ | 所有三个互不接触回路的增益乘积之和 |
| $\cdots$ | 以此类推，更高阶不接触回路组合 |

### 符号规则
- 奇数组不接触回路取负号（$-$），偶数组取正号（$+$）。
- 即：$\Delta = 1 - (\text{单回路和}) + (\text{两两不接触回路积和}) - (\text{三个不接触回路积和}) + \cdots$

## 余子式 $\Delta_k$

- $\Delta_k$ 是从 $\Delta$ 中除去与第 $k$ 条前向通路相接触的所有回路后的剩余部分。
- 若第 $k$ 条前向通路与所有回路都接触，则 $\Delta_k = 1$。
- 若第 $k$ 条前向通路不与任何回路接触，则 $\Delta_k = \Delta$。

## 前向通路

- 前向通路是从源点到阱点，沿支路方向不重复经过任何节点的路径。
- 前向通路增益 $P_k$ 是通路上所有支路增益的乘积。
- 一条信号流图可能有多条前向通路，需要逐一求出。

## 不接触回路

- 两个回路之间没有公共节点时，称为不接触回路。
- 计算 $\Delta$ 时，必须正确识别所有的不接触回路组合。
- 不接触回路可能有多组，需将所有组合的增益乘积相加。

## 计算步骤

1. **列出所有前向通路**：找出从输入到输出的所有路径，计算增益 $P_k$。
2. **列出所有回路**：找出流图中所有闭合回路，计算增益 $L_a$。
3. **找出不接触回路**：确定哪些回路互不接触。
4. **计算 $\Delta$**：按 $\Delta = 1 - \sum L_a + \sum L_b L_c - \cdots$ 展开。
5. **计算 $\Delta_k$**：对每条前向通路，除去接触回路后计算余子式。
6. **代入公式**：$H = \frac{1}{\Delta} \sum P_k \Delta_k$。

## 示例

对于具有一条前向通路 $P_1$ 和两个回路 $L_1$、$L_2$（不接触）的流图：
- $\Delta = 1 - (L_1 + L_2) + L_1 L_2$
- 若 $L_1$ 与 $P_1$ 接触，$L_2$ 不与 $P_1$ 接触，则 $\Delta_1 = 1 - L_2$
- $H = \frac{P_1 (1 - L_2)}{1 - (L_1 + L_2) + L_1 L_2}$
