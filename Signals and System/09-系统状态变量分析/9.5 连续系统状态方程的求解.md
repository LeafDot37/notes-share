# 连续系统状态方程的求解

## 时域法

### 零输入响应与零状态响应

状态方程 $\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{f}$ 的全解：

$$\mathbf{x}(t) = e^{\mathbf{A}(t-t_0)}\mathbf{x}(t_0) + \int_{t_0}^{t} e^{\mathbf{A}(t-\tau)}\mathbf{B}\mathbf{f}(\tau)d\tau$$

- **零输入响应**：$e^{\mathbf{A}(t-t_0)}\mathbf{x}(t_0)$，仅由初始状态引起。
- **零状态响应**：$\int_{t_0}^{t} e^{\mathbf{A}(t-\tau)}\mathbf{B}\mathbf{f}(\tau)d\tau$，仅由输入引起。

### 输出响应
$$\mathbf{y}(t) = \mathbf{C}\mathbf{x}(t) + \mathbf{D}\mathbf{f}(t)$$
$$= \mathbf{C}e^{\mathbf{A}(t-t_0)}\mathbf{x}(t_0) + \mathbf{C}\int_{t_0}^{t} e^{\mathbf{A}(t-\tau)}\mathbf{B}\mathbf{f}(\tau)d\tau + \mathbf{D}\mathbf{f}(t)$$

## 状态转移矩阵 $\mathbf{\Phi}(t) = e^{\mathbf{A}t}$

### 定义
状态转移矩阵 $e^{\mathbf{A}t}$ 描述了系统在没有输入时状态随时间的演化：
$$\mathbf{x}(t) = e^{\mathbf{A}(t-t_0)}\mathbf{x}(t_0)$$

### 性质
| 性质 | 表达式 |
|------|--------|
| 初始值 | $e^{\mathbf{A} \cdot 0} = \mathbf{I}$ |
| 组合性 | $e^{\mathbf{A}t_1} e^{\mathbf{A}t_2} = e^{\mathbf{A}(t_1+t_2)}$ |
| 可逆性 | $(e^{\mathbf{A}t})^{-1} = e^{-\mathbf{A}t}$ |
| 微分 | $\frac{d}{dt} e^{\mathbf{A}t} = \mathbf{A}e^{\mathbf{A}t} = e^{\mathbf{A}t}\mathbf{A}$ |

### 计算方法
- **拉普拉斯变换法**：$e^{\mathbf{A}t} = \mathcal{L}^{-1}\{(\mathbf{sI} - \mathbf{A})^{-1}\}$
- **级数展开法**：$e^{\mathbf{A}t} = \mathbf{I} + \mathbf{A}t + \frac{1}{2!}\mathbf{A}^2 t^2 + \cdots$
- **对角化法**：若 $\mathbf{A} = \mathbf{P}\mathbf{\Lambda}\mathbf{P}^{-1}$，则 $e^{\mathbf{A}t} = \mathbf{P}e^{\mathbf{\Lambda}t}\mathbf{P}^{-1}$

## 拉普拉斯变换法（变换域法）

对方程 $\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{f}$ 两边取拉普拉斯变换：

$$\mathbf{X}(s) = (\mathbf{sI} - \mathbf{A})^{-1}\mathbf{x}(0) + (\mathbf{sI} - \mathbf{A})^{-1}\mathbf{B}\mathbf{F}(s)$$

其中 $(\mathbf{sI} - \mathbf{A})^{-1}$ 称为预解矩阵（resolvent matrix）。

### 求解步骤
1. 计算预解矩阵 $(\mathbf{sI} - \mathbf{A})^{-1}$。
2. 代入初始状态 $\mathbf{x}(0)$ 和输入 $\mathbf{F}(s)$。
3. 得到 $\mathbf{X}(s)$ 后进行拉普拉斯逆变换求得 $\mathbf{x}(t)$。

## 系统函数矩阵

### 定义
从输入 $\mathbf{f}(t)$ 到输出 $\mathbf{y}(t)$ 的传输特性：

$$\mathbf{H}(s) = \mathbf{C}(\mathbf{sI} - \mathbf{A})^{-1}\mathbf{B} + \mathbf{D}$$

### 性质
- $\mathbf{H}(s)$ 是 $p \times m$ 矩阵，元素 $H_{ij}(s)$ 表示第 $j$ 个输入到第 $i$ 个输出的转移函数。
- 系统函数矩阵与状态变量选择无关，是系统的固有特性。
- $\mathbf{H}(s)$ 的极点即为 $\mathbf{A}$ 矩阵的特征值。
