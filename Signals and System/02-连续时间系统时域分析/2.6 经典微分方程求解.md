# 经典微分方程求解

## 微分方程的建立

对于连续时间线性时不变（LTI）系统，其输入-输出关系通常由常系数线性微分方程描述。建立微分方程的依据：

1. **电路元件特性**
   - 电阻：$v(t) = R \cdot i(t)$
   - 电容：$i(t) = C \frac{dv(t)}{dt}$，或 $v(t) = \frac{1}{C} \int i(t) dt$
   - 电感：$v(t) = L \frac{di(t)}{dt}$，或 $i(t) = \frac{1}{L} \int v(t) dt$

2. **基尔霍夫定律**
   - KCL（基尔霍夫电流定律）：流入节点的电流之和等于流出节点的电流之和
   - KVL（基尔霍夫电压定律）：闭合回路中电压降的代数和为零

通过上述定律，可将电路中的电压电流关系表示为关于响应（输出电压）与激励（输入电压）的 $n$ 阶线性微分方程，其一般形式为：

$$
a_n \frac{d^n y(t)}{dt^n} + a_{n-1} \frac{d^{n-1} y(t)}{dt^{n-1}} + \cdots + a_1 \frac{dy(t)}{dt} + a_0 y(t) = b_m \frac{d^m x(t)}{dt^m} + b_{m-1} \frac{d^{m-1} x(t)}{dt^{m-1}} + \cdots + b_1 \frac{dx(t)}{dt} + b_0 x(t)
$$

其中 $y(t)$ 为系统响应，$x(t)$ 为激励信号。

## 经典法求解

经典法直接求解微分方程，其核心思想是将全解分解为**齐次解**（自然响应）与**特解**（强迫响应）之和：

$$
y(t) = y_h(t) + y_p(t)
$$

### 齐次解

齐次解是齐次方程（令激励 $x(t)=0$）的解，由系统的**特征方程**决定。

1. 写出齐次方程：
   $$ a_n \frac{d^n y(t)}{dt^n} + a_{n-1} \frac{d^{n-1} y(t)}{dt^{n-1}} + \cdots + a_1 \frac{dy(t)}{dt} + a_0 y(t) = 0 $$

2. 写出特征方程（将 $\frac{d^k}{dt^k}$ 替换为 $\lambda^k$）：
   $$ a_n \lambda^n + a_{n-1} \lambda^{n-1} + \cdots + a_1 \lambda + a_0 = 0 $$

3. 求解特征根 $\lambda_1, \lambda_2, \ldots, \lambda_n$，根据特征根的类型确定齐次解的形式：

#### （1）单实根
若特征根 $\lambda$ 为单实根，对应齐次解项为：
$$ C e^{\lambda t} $$

#### （2）共轭复根
若特征根为共轭复根 $\lambda_{1,2} = \alpha \pm j\beta$，对应齐次解项为：
$$ e^{\alpha t} \left( C_1 \cos \beta t + C_2 \sin \beta t \right) $$
或等价形式：
$$ A e^{\alpha t} \cos(\beta t + \theta) $$

#### （3）$r$ 重实根
若特征根 $\lambda$ 为 $r$ 重根，对应齐次解项为：
$$ \left( C_{r-1} t^{r-1} + C_{r-2} t^{r-2} + \cdots + C_1 t + C_0 \right) e^{\lambda t} $$

### 特解

特解是满足非齐次方程的一个特解，其函数形式由**激励信号 $x(t)$ 的形式**决定。将特解代入原方程可求得待定系数。

常见激励信号对应的特解形式：

| 激励 $x(t)$ | 特解形式 $y_p(t)$ |
|---|---|
| 常数 $A$ | $B$ |
| $t^p$ | $B_0 t^p + B_1 t^{p-1} + \cdots + B_p$ |
| $e^{\alpha t}$（$\alpha$ 不是特征根） | $B e^{\alpha t}$ |
| $e^{\alpha t}$（$\alpha$ 是 $r$ 重特征根） | $B t^r e^{\alpha t}$ |
| $\cos \beta t$ 或 $\sin \beta t$ | $B_1 \cos \beta t + B_2 \sin \beta t$ |
| $e^{\alpha t} \cos \beta t$ | $e^{\alpha t} \left( B_1 \cos \beta t + B_2 \sin \beta t \right)$ |

### 全解

全解为齐次解与特解之和：
$$ y(t) = y_h(t) + y_p(t) $$

齐次解中的待定系数 $C_1, C_2, \ldots$ 根据**初始条件**（或边界条件）确定。对于 $n$ 阶系统，需要 $n$ 个初始条件（通常为 $y(0), y'(0), \ldots, y^{(n-1)}(0)$）来唯一确定全解。

## 求解步骤总结

1. 根据电路列写微分方程
2. 求特征根，写出齐次解形式（含待定系数）
3. 根据激励形式设定特解，代入原方程求待定系数，得到特解
4. 全解 = 齐次解 + 特解
5. 利用初始条件确定齐次解中的待定系数
6. 得到系统的完全响应
