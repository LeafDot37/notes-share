# 频分复用与时分复用

## 复用技术概述

复用（Multiplexing）技术利用单一物理信道同时传输多路信号，其核心思想是在发送端将多路信号合并为一路复合信号，在接收端再将其分离为各自独立的信号。频分复用（FDM）和时分复用（TDM）是两种最基本的复用方式，分别从频率和时间两个维度对信道资源进行划分。

## 频分复用（FDM）

### 基本原理

频分复用（Frequency Division Multiplexing, FDM）将信道的总带宽划分为多个互不重叠的子频带，每个子频带分配给一路信号。各路信号通过调制到不同载波频率上，在频域上实现正交排列。

FDM 的数学模型：
- 第 $i$ 路基带信号 $f_i(t)$ 调制到载波 $\cos(\omega_i t)$ 上
- 已调信号：$s_i(t) = f_i(t) \cos(\omega_i t)$
- 复合信号：$s_{\text{FDM}}(t) = \sum_{i=1}^{N} s_i(t)$

### 频谱搬移与频率分配

FDM 的核心是频谱搬移。各路基带信号具有相似的频谱分布（如语音信号约 300 Hz 至 3400 Hz），通过调制将它们搬移到不同频率位置：

- 每路信号占据一个子频带（Channel），子频带之间留有保护带（Guard Band）防止串扰
- 保护带的宽度取决于滤波器的选择性和信道稳定性
- 子频带在频域上依次排列，覆盖整个信道带宽

### FDM 系统的组成

**发送端**：
1. 各路基带信号分别经过低通滤波器（带限）
2. 调制到各自的载波频率
3. 所有已调信号相加形成复合信号
4. 必要时进行总功率放大

**接收端**：
1. 复合信号通过带通滤波器组，分离各路信号
2. 每路信号经同步解调恢复基带信号
3. 解调后的信号通过低通滤波器滤除高频分量

### FDM 的优缺点

**优点**：
- 信道利用率高，适合多路信号长期稳定传输
- 各路信号独立传输，互不影响
- 不需要收发端严格的时间同步
- 技术成熟，设备标准化

**缺点**：
- 需要保护带，频谱利用率不是最优
- 对滤波器要求高（陡峭的过渡带）
- 信道非线性会引入互调失真
- 设备体积较大（模拟滤波器难以小型化）

### FDM 的典型应用

- **广播系统**：AM/FM 广播中不同电台使用不同频率
- **电视广播**：每个电视频道占用 6-8 MHz 带宽
- **载波通信系统**：将多路电话信号复用到同一条线路上传输
- **OFDM**：正交频分复用，是 4G/5G 和 Wi-Fi 的核心技术

## 时分复用（TDM）

### 基本原理

时分复用（Time Division Multiplexing, TDM）将时间划分为若干个等长的时隙（Time Slot），每个时隙分配给一路信号。各路信号轮流使用信道的全部带宽，在时间上互不重叠。

TDM 的数学模型：
- 复合信号由各路信号的抽样值按时间轮流排列组成
- 第 $i$ 路信号在第 $i$ 个时隙传输：$s_{\text{TDM}}(t) = \sum_i f_i(t) \cdot p(t - iT_s)$

### 时隙分配与帧结构

TDM 系统的时序结构：
- **帧（Frame）**：一个完整的循环周期，包含所有各路时隙
- **时隙（Time Slot）**：分配给某一路信号的时间段，宽度为 $T_s/N$
- **帧同步**：每个帧开始处通常有特殊的同步码或同步位

典型帧结构：
$$
\boxed{\text{同步}} | \text{时隙 1} | \text{时隙 2} | \cdots | \text{时隙 N} | \boxed{\text{同步}} | \cdots
$$

### TDM 与抽样定理的关系

时分复用建立在抽样定理的基础上。设每路信号的最高频率为 $f_m$，抽样频率必须满足 $f_s \geq 2f_m$。对于 $N$ 路 TDM 系统：

- 每路抽样周期：$T_s = 1/f_s$
- 每路时隙宽度：$T_{\text{slot}} = T_s/N$
- 总传输速率：$R = N f_s$（每路保证奈奎斯特率）

TDM 对信道的带宽要求比单路传输高 $N$ 倍——这是用带宽换取多路能力的体现。

### TDM 的优缺点

**优点**：
- 不需要保护频带，频谱利用率高
- 数字系统中实现简单，易于集成
- 不存在互调失真问题
- 对非线性的容忍度比 FDM 高

**缺点**：
- 需要精确的时间同步
- 对时钟抖动敏感
- 不适合连续波信号（需先抽样数字化）
- 单路用户占用全部带宽速率，设计约束不同

### TDM 的典型应用

- **PCM 电话系统**：E1（2.048 Mbps，32 路）和 T1（1.544 Mbps，24 路）
- **数字通信系统**：SDH/SONET 光传输网络
- **总线系统**：计算机总线的分时访问（如时间片轮转）
- **多路复用 ADC**：单个 ADC 通过模拟开关轮流采样多路信号

## FDM 与 TDM 的比较

| 特性 | 频分复用（FDM） | 时分复用（TDM） |
|------|----------------|----------------|
| 资源划分维度 | 频率 | 时间 |
| 滤波器要求 | 高（带通滤波器组） | 低（时间门控） |
| 时间同步 | 不需要严格同步 | 需要精确同步 |
| 信道非线性影响 | 大（互调失真） | 小 |
| 数字化集成度 | 较低 | 高 |
| 保护开销 | 保护频带 | 保护时间/同步码 |
| 频谱效率 | 较低（保护带开销） | 较高 |
| 主要应用 | 模拟通信系统 | 数字通信系统 |

## 混合复用技术

现代通信系统常结合多种复用方式以进一步提高信道利用率：

- **FDM + TDM**：如 OFDMA（正交频分多址），在频率正交的子载波上再进行时隙分配
- **WDM（波分复用）**：光纤通信中的"FDM"，不同信号使用不同波长的激光

现代 4G/5G 移动通信系统采用的 OFDMA 正交频分多址结合了 FDM 和 TDM 的优势：在频域上使用正交子载波（无需保护带），在时域上为每个用户分配特定的时频资源块，实现了灵活高效的多用户接入。
