# 傅里叶变换的性质

傅里叶变换具有一系列重要的性质，利用这些性质可以高效地推导复杂信号的频谱，而无需反复使用定义式积分。这些性质揭示了时域和频域之间的对偶关系。

## 线性性质

若 $f₁(t) ↔ F₁(ω)$，$f₂(t) ↔ F₂(ω)$，则对于任意常数 a、b：

$$a f_1(t) + b f_2(t) \longleftrightarrow a F_1(\omega) + b F_2(\omega)$$

线性性质是傅里叶变换作为线性算子的本质体现，允许我们将复杂信号分解为基本信号之和，分别变换后再叠加。

## 对称性（对偶性质）

若 $f(t) ↔ F(ω)$，则：

$$F(t) \longleftrightarrow 2\pi f(-\omega)$$

如果 $f(t)$是偶函数，则 $F(t) ↔ 2π f(ω)$。

**意义：** 傅里叶变换的对偶性表明，时域和频域的角色是可以互换的。如果一个信号的时域波形与另一个信号的频谱形状相同，那么这两个信号的傅里叶变换形式上也具有对应关系。

**应用示例：**已知 $δ(t) ↔ 1$，由对称性得 $1 ↔ 2πδ(ω)$。已知矩形脉冲 $↔ Sa$函数，则 Sa脉冲 $↔$矩形频谱。

## 奇偶虚实性

对于实信号 $f(t)$，其傅里叶变换 $F(ω)$具有以下对称特性：

$- F(ω)$的实部 $R(ω)$是偶函数：$R(ω) = R(-ω)$
$- F(ω)$的虚部 $X(ω)$是奇函数：$X(ω) = -X(-ω)$
-幅度谱 $|F(ω)|$是偶函数：$|F(ω)| = |F(-ω)|$
-相位谱 $φ(ω)$是奇函数：$φ(ω) = -φ(-ω)$

**特殊情况：**
-若 $f(t)$为实偶函数，则 $F(ω)$为实偶函数（虚部为零）
-若 $f(t)$为实奇函数，则 $F(ω)$为纯虚奇函数（实部为零）

这一性质在信号处理中很有用——只要知道正频率部分，负频率部分可由对称性完全确定。

## 时移特性

若 $f(t) ↔ F(ω)$，则：

$$f(t - t_0) \longleftrightarrow F(\omega) e^{-j\omega t_0}$$

**物理意义：**信号在时域中延时$t_0$，其在频域中幅度谱不变，相位谱增加一个与频率成正比的相移 $-ωt₀$（即相位在频域中呈线性变化）。

同样地，时域超前则对应正相移：

$$f(t + t_0) \longleftrightarrow F(\omega) e^{j\omega t_0}$$

## 频移特性（调制定理）

若 $f(t) ↔ F(ω)$，则：

$$f(t) e^{j\omega_0 t} \longleftrightarrow F(\omega - \omega_0)$$

$$f(t) \cos(\omega_0 t) \longleftrightarrow \frac{1}{2} [F(\omega - \omega_0) + F(\omega + \omega_0)]$$

$$f(t) \sin(\omega_0 t) \longleftrightarrow \frac{1}{2j} [F(\omega - \omega_0) - F(\omega + \omega_0)]$$

**物理意义：**频移特性是实现调制（Modulation）的理论基础。将信号乘以 $cos(ω₀t)$或 $sin(ω₀t)$（载波），相当于将信号的频谱搬移到 $±ω₀$处。这就是幅度调制（AM）的核心原理。

## 尺度变换特性

若 $f(t) ↔ F(ω)$，则对于非零实常数 a：

$$f(at) \longleftrightarrow \frac{1}{|a|} F\left(\frac{\omega}{a}\right)$$

**物理意义：**
- 时域压缩（|a| > 1）导致频域展宽，幅度降低
- 时域展宽（|a| < 1）导致频域压缩，幅度增大
-时域反转（$a = -1$）对应频域反转：$f(-t) ↔ F(-ω)$

**重要推论：** 信号的时宽-带宽积为常数。时域越窄的信号，频域越宽；时域越宽的信号，频域越窄。这一性质在通信系统的时频分析中具有根本性的意义。

## 时域微分特性

若 $f(t) ↔ F(ω)$，且 $f(t)$在 $t → ±∞$时趋于零，则：

$$\frac{d^n f(t)}{dt^n} \longleftrightarrow (j\omega)^n F(\omega)$$

**应用：** 将信号的微分运算转换为频域中的代数乘法，简化微分方程的求解。

**反用：**若 $g(t)$可以表示为某个函数 $f(t)$的导数，则可以先求 $f(t)$的变换再乘以 $jω$。但需要注意反用可能涉及广义函数的复合运算，如 $xδ'(x) = -δ(x)$此类关系。

### 时域积分特性

若 $f(t) ↔ F(ω)$，且 $F(0) = 0$（即信号面积为 0），则：

$$\int_{-\infty}^{t} f(\tau) d\tau \longleftrightarrow \frac{F(\omega)}{j\omega}$$

若 $F(0) ≠ 0$，则需要引入冲激项：

$$\int_{-\infty}^{t} f(\tau) d\tau \longleftrightarrow \pi F(0) \delta(\omega) + \frac{F(\omega)}{j\omega}$$

## 频域微分特性

若 $f(t) ↔ F(ω)$，则：

$$(-jt)^n f(t) \longleftrightarrow \frac{d^n F(\omega)}{d\omega^n}$$

特别地，$n=1$时：

$$t f(t) \longleftrightarrow j \frac{dF(\omega)}{d\omega}$$

**应用：** 当信号乘以 t 时，对应频域中求导。例如：t·e^{-at}u(t) 的变换可通过单边指数变换求导得到。

### 频域积分特性

$$\frac{f(t)}{-jt} \longleftrightarrow \int_{-\infty}^{\omega} F(\Omega) d\Omega$$

**运用原则：** 在利用微分/积分性质时，看到 t·f(t) 的乘法优先使用频域微分；看到 f(t)/t 的形式优先使用频域积分。尽量不反用时域微积分性质处理复合运算，以避免广义函数运算的复杂性。

## 性质总结

| 性质 | 时域 | 频域 |
|------|------|------|
| 线性 | $a f_1(t) + b f_2(t)$ | $a F_1(\omega) + b F_2(\omega)$ |
| 对称性 | $F(t)$ | $2\pi f(-\omega)$ |
| 时移 | $f(t-t_0)$ | $F(\omega)e^{-j\omega t_0}$ |
| 频移 | $f(t)e^{j\omega_0 t}$ | $F(\omega-\omega_0)$ |
| 尺度变换 | $f(at)$ | $\frac{1}{|a|}F(\frac{\omega}{a})$ |
| 时域微分 | $\frac{d^n f}{dt^n}$ | $(j\omega)^n F(\omega)$ |
| 时域积分 | $\int_{-\infty}^{t} f(\tau)d\tau$ | $\frac{F(\omega)}{j\omega} + \pi F(0)\delta(\omega)$ |
| 频域微分 | $t^n f(t)$ | $j^n \frac{d^n F}{d\omega^n}$ |
