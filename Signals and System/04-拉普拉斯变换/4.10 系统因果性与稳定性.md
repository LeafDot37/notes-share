# 系统因果性与稳定性

因果性和稳定性是系统分析中两个最基本也是最重的特性。通过系统函数 $H(s)$ 的收敛域（ROC）和极点分布，可以方便地判断系统的因果性与稳定性。

## 因果系统

### 定义

若系统在任意时刻 $t_0$ 的输出仅取决于 $t_0$ 及 $t_0$ 之前的输入，而与 $t_0$ 之后的输入无关，则称系统为**因果系统**（Causal System）。

数学表达：因果系统的冲激响应满足：

$$
h(t) = 0, \quad t < 0
$$

### 因果系统的判别

根据系统函数 $H(s)$ 的收敛域，因果系统具有以下特征：

1. **收敛域为右半平面**：因果系统的收敛域是最右边极点以右的半平面，即 $\text{Re}[s] > \sigma_0$
2. **$H(s)$ 为有理函数时**：分子的阶次不超过分母的阶次（即 $m \le n$）
3. **拉普拉斯变换形式**：若 $H(s)$ 可展开为 $s$ 的负幂级数形式，则为因果系统

### 因果性验证

**时域验证**：
$$
h(t) = 0, \quad \forall t < 0
$$

**$s$ 域验证**：
- 若 $H(s)$ 是有理函数，且收敛域为 $\text{Re}[s] > \sigma_{\max}$（最右侧极点的右半平面），则系统因果
- 注意：收敛域为右半平面是因果系统的**必要但不充分**条件（严格来说，若 $H(s)$ 是 $s$ 的有理函数，则收敛域为最右侧极点的右半平面等价于因果性）

## 稳定系统

### 定义

若系统对任意有界输入产生有界输出（BIBO 稳定性），则称系统稳定。BIBO 稳定性的充要条件为：

$$
\int_{-\infty}^{\infty} |h(t)| \, dt < \infty
$$

即冲激响应 $h(t)$ **绝对可积**。

### 稳定系统的 $s$ 域判别

**稳定系统的收敛域必须包含虚轴**（$\text{Re}[s] = 0$）。这是因为：

- 当 $s = j\omega$ 时，系统函数退化为频率响应 $H(j\omega)$
- 稳定的系统必须有定义良好的频率响应

### 与极点分布的关系

对于**因果稳定系统**，$H(s)$ 的极点分布满足：

**所有极点必须在左半平面**（$\text{Re}[p] < 0$）

这一结论非常重要，以下是详细分析：

| 极点位置 | 对应 $h(t)$ 模式 | 稳定性 |
|---|---|---|
| 左半平面（$\text{Re}[p] < 0$） | 指数衰减或衰减振荡 | **稳定** |
| 虚轴单极点（$\text{Re}[p] = 0$） | 常数或等幅振荡 | 临界稳定（不稳定） |
| 虚轴重极点 | 增长项（$t$ 乘以等幅振荡） | **不稳定** |
| 右半平面（$\text{Re}[p] > 0$） | 指数增长或增长振荡 | **不稳定** |

### 关于虚轴上极点的重要说明

部分同学会误以为虚轴上极点的系统是稳定的（因为信号没有发散），但从 BIBO 稳定性的严格定义来看：

- 若系统在虚轴上有极点（如 $H(s) = 1/s$），当输入为单位阶跃信号时，输出将无限增长（系统呈现**累积效应**）
- 因此，虚轴上的极点使系统不满足 BIBO 稳定性，系统为**临界稳定**或**不稳定**
- 在工程上，有时将虚轴上有单极点、无重极点的系统称为"边界稳定"，但它们不满足严格的 BIBO 稳定性定义

## 霍尔维茨多项式

### 定义

若多项式 $D(s) = a_n s^n + a_{n-1} s^{n-1} + \cdots + a_0$ 的所有根均位于 $s$ 平面的左半平面（$\text{Re}[s] < 0$），则称 $D(s)$ 为**霍尔维茨多项式**（Hurwitz Polynomial）。

### 霍尔维茨判别准则

霍尔维茨多项式具有以下性质：

1. **所有系数同号**：$D(s)$ 的所有系数 $a_k$ 必须均为正数（或均为负数）
2. **不缺项**：$D(s)$ 不能缺少任何 $s^k$ 项（从 $s^n$ 到 $s^0$ 项系数均非零）
3. **霍尔维茨行列式**：所有霍尔维茨子行列式均大于零

### 系统函数的分母

因果稳定系统的分母多项式 $D(s)$ 必须是霍尔维茨多项式。对于低通系统，$H(s)$ 的分母即为霍尔维茨多项式。

**低通滤波器的系统函数**分母多为霍尔维茨多项式的典型例子，如巴特沃斯、切比雪夫滤波器的分母多项式均满足霍尔维茨准则。

## 因果稳定系统的判定流程

1. **检查因果性**：确认系统的收敛域为右半平面（$h(t) = 0, t < 0$）
2. **检查稳定性**：所有极点位于左半平面
3. **综合判断**：因果稳定系统的收敛域为 $\text{Re}[s] > \sigma_0$ 且 $\sigma_0 < 0$（虚轴在收敛域内）

## 系统类型总结

| 系统类型 | 因果性 | 稳定性 | $H(s)$ 极点位置 | 收敛域 |
|---|---|---|---|---|
| 因果稳定 | 是 | 是 | 均在左半平面 | $\text{Re}[s] > \sigma_0, \sigma_0 < 0$ |
| 因果不稳定 | 是 | 否 | 存在右半平面/虚轴极点 | $\text{Re}[s] > \sigma_0, \sigma_0 \ge 0$ |
| 非因果稳定 | 否 | 是 | 均在左半平面（双边信号） | 带状区域包含虚轴 |
| 非因果不稳定 | 否 | 否 | — | — |

## 拉普拉斯变换与傅里叶变换的关系

对于稳定系统，$H(s)$ 在虚轴上收敛，因此系统的频率响应可以通过 $s = j\omega$ 代入得到：

$$
H(j\omega) = H(s) \big|_{s = j\omega}
$$

这一关系对系统频域分析至关重要。若系统不稳定（虚轴不在收敛域内），则 $H(j\omega)$ 不存在或发散。

## 工程意义

1. **反馈系统设计**：反馈系统的稳定性至关重要，不稳定的反馈系统会产生自激振荡
2. **滤波器设计**：所有实际使用的滤波器都必须是稳定的
3. **控制系统**：稳定是控制系统正常工作的前提，常用极点配置、PID 调节等方法保证稳定性
4. **信号处理**：数字滤波器设计中需保证所有极点位于单位圆内（$z$ 域），对应 $s$ 域左半平面
