# 差分方程的经典求解

求解差分方程的方法有多种，时域经典法是将全响应分解为**齐次解**（自然响应）与**特解**（强迫响应）之和。

## 差分方程的标准形式

给定 $N$ 阶线性常系数差分方程：

$$
\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]
$$

全解形式为：

$$
y[n] = y_h[n] + y_p[n]
$$

其中 $y_h[n]$ 为齐次解（对应齐次方程的通解），$y_p[n]$ 为特解（满足非齐次方程的一个特解）。

## 齐次解的求解

### 特征方程

齐次方程（令激励 $x[n] = 0$）为：

$$
\sum_{k=0}^{N} a_k y[n-k] = 0
$$

设解的形式为 $y_h[n] = C \lambda^n$，代入齐次方程得到**特征方程**：

$$
a_0 \lambda^N + a_1 \lambda^{N-1} + \cdots + a_{N-1} \lambda + a_N = 0
$$

求解得到 $N$ 个特征根 $\lambda_1, \lambda_2, \ldots, \lambda_N$。

### 特征根形式与对应齐次解

#### 单实根

若特征根 $\lambda$ 为单实根，则对应的齐次解项为：

$$
C \lambda^n
$$

#### $r$ 重实根

若特征根 $\lambda$ 为 $r$ 重根，则对应的齐次解项为：

$$
(C_0 + C_1 n + C_2 n^2 + \cdots + C_{r-1} n^{r-1}) \lambda^n
$$

即从 $C\lambda^n$ 扩展为 $n$ 的多项式乘以 $\lambda^n$。

#### 共轭复根

若特征根为一对共轭复根 $\lambda_{1,2} = \alpha \pm j\beta = re^{\pm j\theta}$，则对应的齐次解项为：

$$
r^n (C_1 \cos(\theta n) + C_2 \sin(\theta n))
$$

或等价形式：

$$
A r^n \cos(\theta n + \phi)
$$

其中 $r = \sqrt{\alpha^2 + \beta^2}$，$\theta = \arctan(\beta/\alpha)$。

### 齐次解的组合

将所有特征根对应的齐次解项线性组合，得到完整的齐次解形式：

$$
y_h[n] = \sum_{i=1}^{N} C_i \lambda_i^n \quad (\text{无重根情形})
$$

带待定系数 $C_i$，由边界条件确定。

## 特解的求解

特解 $y_p[n]$ 的函数形式由**激励 $x[n]$ 的形式**决定。将特解代入原差分方程，通过比较系数确定特解中的待定参数。

### 常见激励对应的特解形式

| 激励 $x[n]$ | 特解形式 $y_p[n]$ |
|---|---|
| 常数 $A$ | $B$ |
| $n^p$（$p$ 次多项式） | $B_0 n^p + B_1 n^{p-1} + \cdots + B_p$ |
| $a^n$（$a$ 不是特征根） | $B a^n$ |
| $a^n$（$a$ 是 $r$ 重特征根） | $B n^r a^n$ |
| $\cos(\omega_0 n)$ 或 $\sin(\omega_0 n)$ | $B_1 \cos(\omega_0 n) + B_2 \sin(\omega_0 n)$ |
| $a^n \cos(\omega_0 n)$ | $a^n(B_1 \cos(\omega_0 n) + B_2 \sin(\omega_0 n))$ |

### 特解待定系数的确定

将设定的特解形式代入差分方程，根据方程两边对应项系数相等建立方程组，即可求解出特解中的待定参数。

**示例**：若 $x[n] = 2^n$ 且 $2$ 不是特征根，设 $y_p[n] = B \cdot 2^n$，代入方程后解出 $B$。

## 全解与边界条件

全解为齐次解与特解之和：

$$
y[n] = y_h[n] + y_p[n]
$$

### 初始条件与边界条件的确定

待定系数 $C_i$ 需要通过系统的**初始条件**来确定。对于 $N$ 阶系统，需要 $N$ 个初始条件。

常用方法：
- 已知 $y[0], y[1], \ldots, y[N-1]$ 直接代入
- 或利用系统在 $n < 0$ 时的历史状态递推得到

将 $n = 0, 1, \ldots, N-1$ 分别代入全解表达式，得到关于 $C_i$ 的 $N$ 个线性方程，联立求解。

## 迭代法

迭代法是差分方程最基本的求解方法，适合计算机数值计算。

将差分方程改写为递归形式：

$$
y[n] = \frac{1}{a_0} \left[ \sum_{k=0}^{M} b_k x[n-k] - \sum_{k=1}^{N} a_k y[n-k] \right]
$$

从已知的初始状态开始，逐步递推计算出所有 $y[n]$。

**优点**：简单直接，适合编程实现。
**缺点**：不易得到闭式解析表达式，难以观察系统的本质特性。

## 经典法求解步骤总结

1. 根据系统列写差分方程
2. 写出特征方程，求解特征根，得到齐次解形式（含待定系数）
3. 根据激励信号的形式设定特解，代入差分方程确定特解参数
4. 全解 = 齐次解 + 特解
5. 利用初始条件（边界条件）确定齐次解中的待定系数
6. 得到系统的完全响应 $y[n]$
