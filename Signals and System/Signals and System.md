# Signals and System

> **Signals and System (信号与系统)** — 从时域到变换域，从连续到离散，系统性地研究信号的表示、变换与系统的分析。

## 总览

| 域 | 连续时间 | 离散时间 |
|---|---|---|
| **时域** | 微分方程、卷积 (Ch2) | 差分方程、卷积和 (Ch7) |
| **频域** | 傅里叶变换 (Ch3、Ch5) | DFT/FFT (Ch8) |
| **复频域** | 拉普拉斯变换 (Ch4) | Z变换 (Ch6) |

## 核心脉络

```
时域分析$──\to $傅里叶变换(频域$) ──\to $拉普拉斯变换(复频域)
    │                                      │
    │  (离散化)                             │ (离散化)
    ▼                                      ▼
离散时域$──\to  DFT/FFT($频域$) ──────\to  Z$变换(复频域)
```

- **时域**: 直观, 求解微分/差分方程
- **频域**: 揭示频率成分, 分析滤波/调制
- **复频域**: 扩展变换适用范围, 分析系统稳定性与因果性
