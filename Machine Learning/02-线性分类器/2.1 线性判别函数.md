# 线性判别函数

线性判别函数是线性分类器的核心，它通过输入特征的线性组合来判定样本所属类别。对于二分类问题，线性判别函数的形式为：

$$
g(\mathbf{x}) = \mathbf{w}^T \mathbf{x} + w_0
$$

其中 $\mathbf{w}$ 是权重向量，$w_0$ 是偏置（阈值）。决策规则为：
- 若 $g(\mathbf{x}) > 0$，则判为正类
- 若 $g(\mathbf{x}) < 0$，则判为负类

决策边界 $g(\mathbf{x}) = 0$ 是一个超平面，将特征空间划分为两个区域。
