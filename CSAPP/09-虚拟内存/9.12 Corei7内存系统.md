# Core i7内存系统

Intel Core i7的内存系统是虚拟内存硬件设计的典型代表，集成了复杂的MMU和TLB层次结构。

**关键特性**：
- **四级页表**：使用4级页表结构，支持48位虚拟地址空间。每级页表大小为4KB，每个PTE占8字节。
- **36位物理地址**：支持最大64GB的物理内存。
- **CR3寄存器**：存放第一级页表（页映射表）的物理地址。

**TLB层次结构**：
- **L1 TLB**：分为数据TLB和指令TLB，容量小但速度极快。
- **L2 TLB**：统一的数据和指令TLB，容量更大，处理L1 TLB未命中的情况。
- 命中TLB时，地址翻译可在1-2个时钟周期内完成。

**地址翻译流程**：
1. CPU计算有效地址（虚拟地址）。
2. 查询TLB加速翻译。
3. TLB未命中时，MMU遍历四级页表。
4. 在物理地址计算完成后，查询L1/L2/L3数据缓存。

Core i7的MMU在硬件上自动遍历多级页表，无需软件干预，硬件页表遍历器（hardware page table walker）在TLB未命中时负责查询页表。
