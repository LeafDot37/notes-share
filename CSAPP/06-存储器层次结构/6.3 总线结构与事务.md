# 总线结构与事务

## 总线（Bus）

**总线**是连接 CPU 和内存（以及 I/O 设备）的一组并行导线，负责在组件之间传输地址、数据和控制信号。

## 读事务

CPU 从内存读取数据的完整流程：

1. CPU 将**地址**放到总线上（总线事务开始）
2. 内存读取地址，从 DRAM 中取出数据
3. 内存将**数据**放到总线上
4. CPU 从总线读取数据（总线事务结束）

## 写事务

CPU 向内存写入数据的完整流程：

1. CPU 将**地址**放到总线上
2. CPU 将**数据**放到总线上
3. 内存读取地址和数据，将数据写入 DRAM
4. 内存发送确认信号（总线事务结束）

## 总线结构

现代系统通常有多个总线层次（如 CPU-内存总线、I/O 总线），通过桥接芯片连接，形成存储器层次结构的基础。

## 重要概念

- 总线事务是 CPU 和内存交互的基本单元
- 每次缓存不命中都会触发一次总线事务
- 总线带宽是系统性能的关键限制因素
