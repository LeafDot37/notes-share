# 十进制与二进制小数转换

## 乘2取整法（十进制小数 → 二进制）

将十进制纯小数转换为二进制小数的方法称为**乘2取整法**：

1. 将小数乘以2
2. 取整数部分作为二进制位（0或1）
3. 将剩余的小数部分重复步骤1-2
4. 直到小数部分为0或达到所需精度

### 示例：0.625 → 二进制

```
0.625 × 2 = 1.25 → 整数部分 1（最高位）
0.25  × 2 = 0.50 → 整数部分 0
0.50  × 2 = 1.00 → 整数部分 1（最低位）
```

结果：$0.625_{10} = 0.101_2$

验证：$1 \times 2^{-1} + 0 \times 2^{-2} + 1 \times 2^{-3} = 0.5 + 0 + 0.125 = 0.625$

### 示例：0.1 → 二进制（循环小数）

```
0.1 × 2 = 0.2 → 0
0.2 × 2 = 0.4 → 0
0.4 × 2 = 0.8 → 0
0.8 × 2 = 1.6 → 1
0.6 × 2 = 1.2 → 1
0.2 × 2 = 0.4 → 0  ← 开始循环
...
```

结果：$0.1_{10} = 0.0001100110011..._2$（无限循环小数）

##二进制小数$→$十进制

$$
0.b_1b_2b_3...b_n = \sum_{i=1}^{n} b_i \times 2^{-i}
$$

例如：$0.101_2 = 1 \times 2^{-1} + 0 \times 2^{-2} + 1 \times 2^{-3} = 0.625_{10}$

## 重要的启示

### 十进制小数不一定能精确表示为二进制

这是一个非常重要的计算机科学知识点：

- 分母只含质因数2的十进制小数才能用有限位二进制小数精确表示
  - $0.5_{10} = 0.1_2$（分母为2）
  - $0.25_{10} = 0.01_2$（分母为4 = $2^2$）
  - $0.75_{10} = 0.11_2$（分母为4）

- 分母含其它质因数的十进制小数会变成**二进制无限循环小数**
  - $0.1_{10} = 1/10 = 1/(2 \times 5)$，分母含质因数5，无法用有限二进制小数表示
  - $0.2_{10}$、$0.3_{10}$ 同理

### 实际影响

- 计算机中浮点数无法精确表示 $0.1$、$0.2$ 等常见十进制小数
- 这就是为什么 `0.1 + 0.2 != 0.3`（在二进制浮点数运算中）
- 金融计算应避免使用二进制浮点数（使用定点数或十进制浮点数）
